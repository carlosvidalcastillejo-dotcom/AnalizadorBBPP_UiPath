# Tests

Esta carpeta contiene scripts de prueba y validación del proyecto.

## Archivos de Prueba

Los archivos `test_*.py` son scripts de desarrollo utilizados para:
- Probar funcionalidades específicas durante el desarrollo
- Validar cambios en componentes individuales
- Verificar integraciones entre módulos
- Debug de issues específicos

## Uso

Estos scripts **no son parte del sistema de tests automatizado**. Son herramientas de desarrollo ad-hoc creadas para validar funcionalidades específicas durante el desarrollo.

Para ejecutar un test específico:
```bash
python tests/test_nombre.py
```

**Nota**: Estos tests pueden requerir ajustes según el estado actual del código y no están mantenidos activamente.
